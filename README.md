# 🏏 IPL Data Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the IPL dataset (2008–2022) using Python, Pandas, and Matplotlib. The project explores team performance, player statistics, and match trends through data analysis and visualization.

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib

## 📂 Dataset
- IPL_Matches_2008_2022.csv
- IPL_Ball_by_Ball_2008_2022.csv

## 📊 Analysis Performed
- Matches played each season
- Top 10 Player of the Match winners
- Top 10 Run Scorers
- Top 10 Wicket Takers
- Matches won by Runs vs Wickets
- Mumbai Indians winning matches
- Missing Value Analysis

## 📈 Visualizations
- Bar Charts
- Count Plots
- Player Performance Analysis

## 🎯 Conclusion
The analysis provides insights into IPL match results, batting performance, bowling performance, and team statistics. It demonstrates the use of Python for real-world data analysis.
